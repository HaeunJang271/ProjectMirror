## A–Z / 가나다 순

### 도착자 (Arrival)

| 항목 | 내용 |
|------|------|
| 정의 | 거울성에 깨어난 플레이어. 고유 이름 없음 |
| UI 표기 | "당신" |
| 관련 | 거울, F-001 |
| 금지 | "주인공", "영웅", "선택받은 자" 등 클리셰 호칭 |

---

### 반영 (Reflection)

| 항목 | 내용 |
|------|------|
| 정의 | 거울이 비추는 **이미지**. '나'와 동일시할 수 없음 |
| 게임 내 | 반영은 0~1.2초 지연될 수 있음 (F-003). 성향에 따라 내용 변화 |
| 철학 | Q-002 — 반영은 나인가, 나의 이미지인가 |
| 금지 | "진짜 나", "가짜 나"로 이분법 설명 |

---

### 거울 (Mirror)

| 항목 | 내용 |
|------|------|
| 정의 | 유리가 아닌 **자기를 마주하려는 의지**가 굳어진 존재 |
| 규칙 | 거짓말 안 함 · 모든 진실 말 안 함 · 질문 없이 말 안 함 |
| 종류 | 반사 거울, 대화 거울(Ch3~), 금 간 거울 |
| 상징 | [SymbolDictionary.md](./SymbolDictionary.md) |

---

### 거울성 (Mirror Citadel)

| 항목 | 내용 |
|------|------|
| 정의 | 세계 중심의 거대 구조. **장소이자 존재** |
| 특징 | 스스로 자람. 건축가 없음. 층·방이 필요에 따라 생김 |
| 1층(Ch1) | 기억의 홀 일대 |
| 2층(Ch2) | 잊힌 층 — 비의 복도 |

---

### 금 (Crack)

| 항목 | 내용 |
|------|------|
| 정의 | 거울 표면의 균열. **선택의 흔적** |
| 대표 | F-001 — Ch1 핵심 복선 |
| 의미 | 거짓말이 아님. 숨기지도 않음. 그냥 **있음** |
| 관련 NPC | 세결 — "고치지 마세요" |

---

### 관리인 (Keeper)

| 항목 | 내용 |
|------|------|
| 정의 | 거울성을 **돌본다고 믿어지는** 존재. 공식 직책 아님 |
| 특징 | 사람인지 거울인지 문서상 확정 안 됨 |
| 최초 언급 | Chapter 2 — 소(NPC-006)가 부인함 |
| 관련 | World Bible Keeper(AI 역할)와 혼동 주의 — AI는 설정 검수자 |

---

### 기억 (Memory)

| 항목 | 내용 |
|------|------|
| 정의 | 잡히지 않는 것. 물·서리·이름 글씨로 **은유** |
| 게임 | NPC·거울·세계가 "기억함". 플레이어 일기(대화 로그)만 명시적 |
| 철학 | Q-001, Q-006 |
| 금지 | 기억 복원 = 정답 엔딩 클리셰 |

---

### 도달자 / 질문 (Question)

| 항목 | 내용 |
|------|------|
| 정의 | 게임이 던지는 **답 없는 문장**. ID는 Q-XXX |
| 형태 | 대사, 상징, 선택, 침묵 |
| 예 | Q-002 "거울에 비친 것은 나인가" |
| 금지 | NPC가 철학 교과서처럼 설명 |

---

### 성향 (Disposition)

| 항목 | 내용 |
|------|------|
| 정의 | 선택에 따라 누적되는 **내부 가중치** 8종 |
| 목록 | 공감, 호기심, 질서, 자유, 희생, 자기중심, 의심, 용기 |
| UI | **절대 노출 안 함** |
| 반영 | 색감, BGM, 대사, 거울 |

---

### 복선 (Foreshadow)

| 항목 | 내용 |
|------|------|

| ID | F-XXX |

### 여운 (Resonance)

| 항목 | 내용 |
|------|------|
| 정의 | 씬 종료 후 남는 **이름 붙이기 어려운 감정** |
| 설계 | emotion_designer — 슬픔 대신 허전함 등 |
| 연출 | 침묵, 짧은 문장, 미완 결말 |

### 이름 (Name)

| 항목 | 내용 |
|------|------|
| 정의 | 정체성의 **자리**. 잃어도 자아가 사라지는지는 Q-006 |
| 게임 | 린의 책, 플레이어 이름 선택(기록 여부) |
| 상징 | 꽃 — 이름 없는 질문 |
