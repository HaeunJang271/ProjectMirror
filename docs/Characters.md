| 항목 | 내용 |
|------|------|
| 나이 | 17 (겉보기) |
| 외형 | 옅은 갈색 머리, 큰 눈, 항상 창가 의자에 앉음 |
| 말투 | 느림. 말 끝이 흐려짐 |
| 성격 | 관찰자, 결단 회피, 충동은 있으나 행동 안 함 |

| 좋아하는 것 | 새 소리, 오후 빛 |
| 싫어하는 것 | "왜 안 열어?"라는 질문 |
| 대표 대사 | "밖에 있으면 …아직 날 수 있어요. 열면 …모를 것 같아서." |
