## 핵심 게임 루프

```
┌─────────────┐
│ 플레이어 행동 │  탐색, 대화, 관찰, 터치
└──────┬──────┘
       ▼
┌─────────────┐
│    선택     │  명시적(대화) + 암시적(경로, 침묵, 체류 시간)
└──────┬──────┘
       ▼
┌─────────────┐
│  세계 변화   │  NPC 기억, 문/창 상태, 조명, BGM 미세 변화
└──────┬──────┘
       ▼
┌─────────────┐
│  거울 변화   │  반사 내용, 말투, 금의 형태
└──────┬──────┘
       ▼
┌─────────────┐
│ 플레이어 감정 │  설명 없이 여운만
└──────┬──────┘
       ▼
     (반복)
```

**왜 이 루프인가:** 플레이어가 "내가 바꿨다"는 인식보다 "세계가 달라진 것 같다"는 감각을 주기 위함. 인과를 숨기면 스스로 해석하게 된다.

## 플레이 흐름

| 단계 | 내용 |
|------|------|
| Prologue | 어둠 → 첫 문 → 거울 실루엣 → 거울성 도착 인지 |
| Chapter 1~N | 허브(복도) + 2~4개 씬 탐색 → 핵심 선택 1~2개 → 챕터 전환 |
| 후반 | 복선 회수, 거울과의 대화 증가 |
| 엔딩 | 가치관 반영형 5+1종 |

**진행 방식:** 챕터 기반 선형 + 씬 내 자유 탐색. 맵 백트래킹 제한적(복도 허브).

**저장 방식 (설계):**

| 유형 | 시점 |
|------|------|
| 자동 저장 | 챕터 시작·핵심 선택 직후 |
| 수동 저장 | 메뉴 (슬롯 3개) |
| 회차 데이터 | DataStore — 성향·복선·회차 수 (UI 비노출) |

## 숨겨진 성향 시스템

### 원칙

- 내부 가중치 `-100 ~ +100` (UI **절대 노출 안 함**)
- 선택마다 1~3개 성향에 `+3 ~ +8` (크기도 숨김)
- **지배 성향**이 아닌 **조합**으로 엔딩·연출 결정

### 8성향 정의 및 반영

| 성향 | 내부 키 | 선택 예시 | 세계 반영 |
|------|---------|-----------|-----------|
| 공감 | `empathy` | NPC 말 끝까지 듣기, 세결의 조언 수용 | NPC 대사 톤 부드러워짐, 이름 기억 |
| 호기심 | `curiosity` | 숨은 통로, 거울 뒤 관찰 | 추가 관찰 포인트, 새로운 대사 옵션 |
| 질서 | `order` | 정해진 길, 기록 따르기 | 복도 정돈된 느낌, BGM 리듬 안정 |
| 자유 | `freedom` | 지름길, 창문 쪽 이동 | 열린 문 증가, 새 소리 증가 |
| 희생 | `sacrifice` | 자신의 단서를 NPC에게 | NPC가 플레이어 이름 부름, 복선 조기 회수 |
| 자기중심 | `self` | 거울만 보기, NPC 무시 | 거울 반사 비중 증가, NPC 거리감 |
| 의심 | `doubt` | 거울 말 반복 확인, 선택 지연 | 거울 반사 0.3초 지연, 대사 중복 |
| 용기 | `courage` | 금 간 거울에 손 대기 | 금이 번지지 않음, Chapter 7 분기 |

### 왜 숫자가 없는가

점수가 보이면 게임이 "맞추기"가 된다. Project MIRROR는 **자기 이해**가 목표이지 최적화가 아니다.

## 선택 구조

### 1. 즉각적 영향 (0~30초)

- 대사 1~2줄 변화
- 효과음 1종
- 조명 미세 변화

### 2. 장기적 영향 (같은 챕터 내)

- NPC 추가 대화 옵션
- 씬 출입 가능 여부
- 오브젝트 상태 (꽃, 창문)

### 3. 누적 영향 (챕터 간)

- 성향 가중치
- 복선 심기/회수
- 거울 반사 콘텐츠

### 4. 회차 영향 (2회차+)

- Prologue 거울 실루엣이 다름
- 일부 NPC가 플레이어를 "기억함"
- 숨겨진 씬 1~2개

### 기억 레이어

| 레이어 | 저장 내용 | 플레이어 인지 |
|--------|-----------|---------------|
| 플레이어 기억 | 대화 로그 (메뉴 열람 가능) | 직접 |
| NPC 기억 | 선택 플래그 | 간접 (대사 변화) |
| 거울 기억 | 반사 이력, 질문 기록 | 간접 (거울 대사) |
| 세계 기억 | 환경 상태 전역 | 간접 (분위기) |

## Chapter 1 핵심 선택: 금이 간 거울

**위치:** LOC-004 금이 간 홀
**철학 질문:** Q-002 — 거울에 비친 것은 나인가, 나의 이미지인가?
**복선:** F-001 심기

| 선택 | 성향 | 즉각 | 장기 | 복선 |
|------|------|------|------|------|
| A. 손을 댄다 | 용기+, 호기심+ | 손끝 차가움, 금이 살짝 번짐 | 세결이 나중에 다른 대사 | F-001: 금 번짐 패턴 저장 |
| B. 지나친다 | 질서+, 의심+ | 발소리만, 침묵 2초 | 린이 "그 거울 앞에선 멈추지 마" | F-001: 금 유지 |
| C. 거울을 등진다 | 공감+, 자유+ | 거울 속 자신만 바라봄 | 가온 창가 새가 날아감 | F-001: 금에서 빛 한 줄 |

**게임은 이유를 설명하지 않음.**

## 엔딩 시스템

### 원칙

좋음/나쁨 없음. **플레이어 가치관의 거울**.

### 엔딩 5+1 (초안)

| ID | 명칭 | 지배 조합 (내부) | 플레이어가 가져가는 것 |
|----|------|------------------|------------------------|
| E-01 | 반사 | 공감+희생 | 타인과 나의 경계에 대한 질문 |
| E-02 | 균열 | 용기+의심 | 진실을 향한 불편함 |
| E-03 | 정적 | 질서+의심 | 선택하지 않음의 무게 |
| E-04 | 열림 | 자유+호기심 | 아직 끝나지 않은 가능성 |
| E-05 | 잔상 | 자기중심+질서 | 나만 남았을 때의 고독 |
| E-06 | 물음 (숨김) | 2회차+, F-001~F-020 전부 회수 | "질문이 답보다 길다" |

분기 조건은 **단일 성향 MAX**가 아니라 **상위 2종 조합** + 핵심 복선 회수 여부.

## UI 컨셉

| 요소 | 설계 |
|------|------|
| HUD | 없음에 가깝게 — 체력·점수·성향 바 없음 |
| 대화 | 하단 텍스트, 이름 표시, 선택지 2~3개 |
| 변화 피드백 | 색온도, Vignette, BGM 레이어, 거울 테두리 |
| 메뉴 | 일기(대화 로그), 설정, 저장만 |
| 접근성 | 자막 크기, 대비, 탭 홀드 시간 조절 |

### 변화를 느끼게 하는 연출 (숫자 대신)

| 신호 | 성향 예시 |
|------|-----------|
| 복도 벽 색온도 | 공감↑ → 따뜻 / 의심↑ → 차가움 |
| BGM 레이어 | 호기심↑ → 고음 단계 추가 |
| 거울 프레임 색 | 용기↑ → 금색 테두리 미세 |
| NPC 눈 맞춤 | 희생↑ → 시선 증가 |

## 게임 규칙

1. 거울에게 거짓말할 수 없음 (플레이어 발화는 제한 없음)
2. 한 챕터에서 핵심 선택은 최대 2개
3. 되돌리기 없음 (챕터 내 자동저장 후)
4. NPC는 죽지 않음 — 사라지거나 기억을 잃을 수는 있음
5. 전투 없음
