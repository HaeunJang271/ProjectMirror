# Project MIRROR — UI Flow

> 화면 흐름·UX 설계. 유지: `prompts/gameplay_designer.md`, `prompts/art_director.md`
> **v0.1**

---

## UX 원칙

1. **HUD 최소** — 체력·성향·점수 없음
2. **텍스트 중심** — 조용한 읽기 경험
3. **변화는 느끼게** — 색·소리·침묵으로만
4. **되돌리기 없음** — 선택 후 자동 저장
5. **접근성** — 자막·대비·탭 홀드

---

## 화면 목록

| ID | 화면명 | 용도 |
|----|--------|------|
| SCR-TITLE | 타이틀 | 시작·이어하기·설정 |
| SCR-NEW | 신규 확인 | "기억이 이어지지 않을 수 있음" |
| SCR-LOAD | 불러오기 | 슬롯 3 |
| SCR-NARR | 내러티브 | Prologue 독백·연출 |
| SCR-EXPLORE | 탐색 | 허브·씬 이동 |
| SCR-DIALOGUE | 대화 | NPC·거울 |
| SCR-CHOICE | 선택 오버레이 | 명시적 분기 |
| SCR-JOURNAL | 일기 | 대화 로그 (성향 비표시) |
| SCR-SETTINGS | 설정 | 자막·음량·저장 |
| SCR-PAUSE | 일시정지 | 저널·설정·타이틀 |
| SCR-CHAPTER | 챕터 전환 | 제목 카드만, 3초 |

## 앱 전체 흐름

```
[SCR-TITLE]
    │
    ├─ 새 이야기 ──▶ [SCR-NEW] ──▶ [SCR-NARR Prologue]
    │
    ├─ 이어하기 ──▶ [SCR-LOAD] ──▶ [마지막 씬]
    │
    └─ 설정 ──▶ [SCR-SETTINGS]

[Prologue P-01..P-06]
    │
    ▼
[SCR-CHAPTER "첫 반사"]
    │
    ▼
[SCR-EXPLORE 허브: 기억의 홀]
    │
    ├─ 씬 진입 ──▶ [SCR-DIALOGUE] / [SCR-NARR]
    │
    ├─ 핵심 선택 ──▶ [SCR-CHOICE] ──▶ 자동저장
    │
    └─ 챕터 종료 ──▶ [SCR-CHAPTER "비의 자리"] ──▶ Ch2
```

## Chapter 1 UX

### 허브: 기억의 홀

```
┌─────────────────────────────────────┐
│  [이름의 서재]    [닫힌 창가]          │
│       │               │              │
│       └───────┬───────┘              │
│               │                      │
│         [플레이어]                    │
│               │                      │
│       [금이 간 홀] ────▶ [Ch2 문]    │
└─────────────────────────────────────┘
        상단: 햄버거 → 일기·설정·저장
        하단: 없음 (HUD 없음)
```

### 대화 화면 레이아웃

```
┌─────────────────────────────────────┐
│         (배경 일러스트/영상)           │
│                                     │
├─────────────────────────────────────┤
│  린                                  │
│  "이름은 적어두면 잊지 않아요."        │
│                                     │
│  ┌─────────────────────────────┐    │
│  │ 이름을 적어줄게              │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │ 그냥 지나친다                │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

- 선택지 2~3개. 4개 이상 금지 (피로도)
- 선택 후 1.5초 침묵 → 자동저장 토스트 **없음**

## 피드백 (숫자 대신)

| 신호 | UX 표현 |
|------|---------|
| 성향 변화 | 0.5초 색온도 트랜지션 |
| 중요 선택 | 2초 침묵 + 화면 미세 진동(옵션) |
| 복선 심기 | 없음 — 플레이어 모름 |
| 자동저장 | 표시 안 함 |

## 접근성

|------|------|
