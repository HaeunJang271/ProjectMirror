# Project MIRROR — Symbol Dictionary

## 상징 관리 규칙

각 상징 항목은 다음을 포함해야 합니다.

- **의미** (Project MIRROR 맥락에서의 의미 — 범용 상징과 다를 수 있음)
- **첫 등장**
- **재등장**
- **엔딩 연결**
- **관련 상징**
