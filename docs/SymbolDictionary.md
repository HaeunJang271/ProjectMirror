# Project MIRROR — Symbol Dictionary

> 모든 상징은 `prompts/symbol_designer.md`가 관리합니다.
> **v0.2** — Sprint 1: 핵심 상징 10종 확정

## 상징 관리 규칙

각 상징 항목은 다음을 포함합니다.

- **의미** — Project MIRROR 맥락 (범용 상징과 다를 수 있음)
- **첫 등장** · **재등장** · **엔딩 연결** · **관련 상징**
- **범용 의미와의 차이** — 의도적 이탈 기록

## 핵심 상징 10종

### 거울 (Mirror)

| 항목 | 내용 |
|------|------|
| 의미 | **자기를 마주하려는 의지**가 굳어진 형태. 유리가 아님 |
| 범용과의 차이 | 진실의 창 ❌ → 질문을 되돌려주는 존재 ✅ |
| 첫 등장 | Prologue — 움직임이 한 박자 늦은 실루엣 |
| 재등장 | 전 챕터. 형태·말투·반사 내용이 성향에 따라 변화 |
| 엔딩 연결 | 플레이어가 마지막에 거울과 어떤 관계를 맺는지가 엔딩 톤 결정 |
| 관련 상징 | 빛, 물, 문 |

> 규칙: 거짓말하지 않음 · 모든 진실을 말하지 않음 · 질문받지 않은 것은 말하지 않음

---

### 새 (Bird)

| 항목 | 내용 |
|------|------|
| 의미 | **아직 실행되지 않은 선택**. 자유가 아니라 '날 수 있지만 날지 않은 것' |
| 범용과의 차이 | 해방·희망 ❌ → 열리지 않은 가능성 ✅ |
| 첫 등장 | Chapter 1 — 닫힌 창가(LOC-003) 밖 소리 |
| 재등장 | Chapter 3, 5. 창문 개방 여부에 따라 새 수·소리 변화 |
| 엔딩 연결 | "날았는가 / 남았는가"는 엔딩에서 행동의 회고가 됨 |
| 관련 상징 | 창문, 문 |

---

### 꽃 (Flower)

| 항목 | 내용 |
|------|------|
| 의미 | **아직 답하지 않은 질문**. 피어났지만 이름 붙이지 않은 것 |
| 범용과의 차이 | 희망·사랑·순수 ❌ → 미완의 질문 ✅ |
| 첫 등장 | Chapter 1 — 이름의 서재(LOC-002) 창틀 틈 파란 꽃 1송이 |
| 재등장 | 챕터마다 색·수 변화 (성향 반영, 수치 노출 없음) |
| 엔딩 연결 | 엔딩 직전 홀에 꽃이 피었는지·시들었는지·몇 송이인지 |
| 관련 상징 | 색, 물, 빛 |

> 파란 꽃 = 희망이 아님. "이름 없는 질문 하나"

---

### 물 (Water)

| 항목 | 내용 |
|------|------|
| 의미 | **잡히지 않는 기억**. 비침은 있으나 손에 남지 않음 |
| 범용과의 차이 | 정화·생명 ❌ → 흐르고 사라지는 잔상 ✅ |
| 첫 등장 | Chapter 1 — 기억의 홀(LOC-001) 바닥 얕은 웅덩이 |
| 재등장 | Chapter 2(비), Chapter 4(거울 액체), Chapter 6 |
| 엔딩 연결 | 마지막 거울 앞 물 웅덩이에 무엇이 비치는가 |
| 관련 상징 | 거울, 비, 달 |

---

### 달 (Moon)

| 항목 | 내용 |
|------|------|
| 의미 | **말하지 않은 진실**. 비추지만 닿을 수 없는 것 |
| 범용과의 차이 | 낭만·신비 ❌ → 거리 두고 보는 인식 ✅ |
| 첫 등장 | Prologue 끝 — 창 너머 희미한 달빛 |
| 재등장 | 야간 씬, Chapter 5·7. 달 모양(초승→보름)이 회차에 따라 변화 |
| 엔딩 연결 | 엔딩 크레딧 전 달빛 아래 거울성 실루엣 |
| 관련 상징 | 빛, 창문, 물 |

---

### 비 (Rain)

| 항목 | 내용 |
|------|------|
| 의미 | **씻겨 나간 것의 자리** — 슬픔이 아닌 **허전함** |
| 범용과의 차이 | 우울·눈물 ❌ → 비어 있음의 감각 ✅ |
| 첫 등장 | Chapter 2 (예정) — Chapter 1 끝 창밖 기운으로 예고 |
| 재등장 | Chapter 2, 6, 8 |
| 엔딩 연결 | 비 온 뒤 거울에 남는 것(얼룩/맑음) |
| 관련 상징 | 물, 창문 |

> emotion_designer: "슬픔을 주지 말고 허전함을 남겨라"

---

### 창문 (Window)

| 항목 | 내용 |
|------|------|
| 의미 | **보는 것과 가는 것 사이의 경계**. 넘지 않은 시선 |
| 범용과의 차이 | 기회·해방 ❌ → 아직 열지 않은 질문 ✅ |
| 첫 등장 | Chapter 1 — 닫힌 창가(LOC-003) |
| 재등장 | 전 챕터. 열림 정도가 누적 선택에 따라 변화 |
| 엔딩 연결 | 마지막 장면 창문 열림/닫힘/부서짐 |
| 관련 상징 | 문, 새, 빛 |

---

### 문 (Door)

| 항목 | 내용 |
|------|------|
| 의미 | **선택 그 자체**. 미지로 통과하는 행위 |
| 범용과의 차이 | 새 출발 ❌ → 되돌릴 수 없는가에 대한 질문 ✅ |
| 첫 등장 | Prologue — 첫 번째 문. 손잡이가 따뜻함 |
| 재등장 | 챕터 전환마다. 문 소리(금속/나무)가 성향에 따라 다름 |
| 엔딩 연결 | 엔딩마다 다른 문이 열림 — 같은 문이 아님 |
| 관련 상징 | 창문, 거울 |

---

### 빛 (Light)

| 항목 | 내용 |
|------|------|
| 의미 | **무엇이 드러나는가** — 노출이 아닌 **인식의 방향** |
| 범용과의 차이 | 진리·구원 ❌ → 어디를 비추느냐 ✅ |
| 첫 등장 | Prologue — 문 틈 새어 나오는 빛 |
| 재등장 | 전 씬. 조명 각도·색온도가 성향 반영 |
| 엔딩 연결 | 엔딩별 빛의 출처(창/거울/달/없음) |
| 관련 상징 | 거울, 색, 달 |

---

### 색 (Color)

| 항목 | 내용 |
|------|------|
| 의미 | **성향의 무의식적 반영**. 감정 라벨이 아님 |
| 범용과의 차이 | 빨강=위험, 파랑=슬픔 ❌ → UI·환경 틴트의 미세 변화 ✅ |
| 첫 등장 | Chapter 1 — 성향 첫 누적 후 복도 색온도 0.5% 변화 (체감 불가 수준) |
| 재등장 | 누적될수록 환경·의상·거울 테두리 색이 서서히 이동 |
| 엔딩 연결 | 엔딩 씬 지배 색상 = 가장 강한 성향 2종의 혼합 |
| 관련 상징 | 꽃, 빛 |

> 플레이어에게 색의 의미를 설명하지 않음
