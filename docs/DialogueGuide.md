# Project MIRROR — Dialogue Guide

> 유지: `prompts/narrative_director.md` | 감정 톤: `prompts/emotion_designer.md`

---

## 대사 원칙

1. **설명하지 않는다** — 플레이어에게 답을 주지 않음
2. **질문한다** — 대사는 질문이거나 질문을 유도함
3. **훈계하지 않는다** — 도덕적 판단 금지
4. **여운을 남긴다** — 완결된 문장보다 열린 문장
